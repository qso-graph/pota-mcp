# pota-mcp

MCP server for [Parks on the Air](https://pota.app/) -- park lookup, activator/hunter stats, spots.

Part of the [qso-graph](https://qso-graph.io/) collection of amateur radio MCP servers.

## Planned Tools

| Tool | Description |
|------|-------------|
| `pota_park_lookup` | Look up a POTA park by reference number |
| `pota_activator_stats` | Activator statistics and park list |
| `pota_hunter_stats` | Hunter statistics and activations worked |
| `pota_spots` | Current POTA spots (live activations) |
| `pota_park_search` | Search parks by name, state, or location |
| `pota_leaderboard` | Top activators/hunters rankings |
